# A minecraft server launcher ment to make it easy for server owners to manage servers

**Goals**

- [ ] Allow for Grouping Backend's into Proxy's (Velocity, Gate, Bungeecord, etc)

- [ ] Plugin/Mod Installer/Manager (Hangar, Modrinth, Spigot)

- [ ] Config Editor

- [ ] Whitelist/Ban Editor

- [ ] Auto Java Checker
